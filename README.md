# password-strength-checker
A password strength checker project using TDD and Git

Password Strength Checker
This project implements a password strength checker using Test-Driven Development (TDD). It includes:

A function in passwords.py that checks password strength.
Unit tests in test_passwords.py using Python's unittest framework.
Documentation files (under the docs folder) related to the sprint (stand-ups, review, retrospective).
How to Run Tests
You can run the tests using Python:

python -m unittest test_passwords.py
